# Forex Data Engineering Pipeline

This project builds a full data pipeline using Kafka, PostgreSQL, FastAPI, Streamlit, and Quelt to process and visualize Forex data in real-time.

## Stack
- EC2 (host)
- Kafka (streaming)
- Python (data flow)
- PostgreSQL (storage)
- FastAPI (backend)
- Streamlit (frontend)
- Quelt (ML)
